### 4. root + www 改用 Cloudflare Redirect Rule

```
chatbot.tw.       A      217.70.184.38           (Gandi webredir IP — 不匯入)
www.chatbot.tw.   CNAME  webredir.vip.gandi.net  (Gandi webredir — 不匯入)
```

→ Cloudflare 沒有 Gandi webredir 等價物。**Cloudflare 設定方法**：

1. **DNS records**（為了讓 redirect rule 能匹配，需有 placeholder）：
   - `A` `@` → `192.0.2.1`（保留 IP，proxy 開橘雲）
   - `CNAME` `www` → `chatbot.tw`（proxy 開橘雲）
2. **Redirect Rule**（Rules → Redirect Rules）：
   - 條件：`(http.host eq "chatbot.tw" or http.host eq "www.chatbot.tw")`
   - Action: Static redirect → `https://www.facebook.com/groups/chatbot.tw/`
   - Status: `301 Permanent`
   - Preserve query string: 視需求

### 5. 沒有 CAA — 任何 CA 都能簽憑證

→ Cloudflare 自動發 SSL 憑證沒問題。完成搬遷後可考慮加：

```
chatbot.tw.   IN  CAA  0 issue "letsencrypt.org"
chatbot.tw.   IN  CAA  0 issue "google.com"
chatbot.tw.   IN  CAA  0 iodef "mailto:你的email"
```

（Cloudflare auto SSL 用 Let's Encrypt 與 Google Trust Services）

### 6. 沒有 SPF / DKIM / DMARC

→ 如有發信需求（不是只有收信 forward）建議補上，但與本次搬遷無關，可後續處理。
